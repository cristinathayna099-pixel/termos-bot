Política de Privacidade - TØRAJØ🍏

Última atualização: 02 de Junho de 2026

Esta Política de Privacidade explica como TØRAJØ🍏 coleta, usa e protege as informações dos usuários e dos servidores do Discord.

1. Dados que Coletamos

O TØRAJØ🍏 pode processar informações públicas fornecidas pela API do Discord, tais como: IDs de usuários, IDs de servidores, nomes de canais e o conteúdo de mensagens enviadas diretamente como comandos para o bot.

2. Como Usamos os Dados

Os dados coletados são utilizados exclusivamente para o funcionamento dos comandos do bot, personalização de configurações no servidor (como sistemas de níveis ou moderação) e melhoria da experiência do usuário. Nenhum dado é vendido ou compartilhado com terceiros.

3. Retenção e Exclusão de Dados

Os dados são armazenados apenas pelo tempo necessário para executar suas funções. Se você desejar que os seus dados ou os dados do seu servidor sejam removidos do nosso banco de dados, entre em contato conosco através do nosso servidor de suporte.

4. Segurança

Adotamos medidas padrão para proteger as informações armazenadas, mas lembramos que nenhum método de transmissão ou armazenamento na internet é 100% seguro.
